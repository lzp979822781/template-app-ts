# 路由方案

路由分为两种原生跳Taro、Taro跳其他

## 原生跳Taro

在同个App这中不同模块可能复用同一个Taro工程，这样涉及到原生工程跳转到不同的Taro页面。但Taro的首页不能动态加载，目前的方案是通过原生页面传参， 在Taro首页属性中根据传入的参数实现前端的跳转，首页设置为一个空页面

## Taro跳转到其他

目前通过jdreact跳转到原生，目前通过openApp协议实现，所以核心也是通过Native modules的jumpToOpenapp实现，下面提供的是jdreact封装的跳转原生的代码，如果不想通过jdreact去做跳转可以自己基于React Native中的NativeModules模块实现跳转，Taro跳转到h5是通过原生中转实现的

```
import { JDJumping } from "@jdreact/jdreact-core-lib";

function jumpToApp(des, params = {}) {
        JDJumping.jumpToOpenapp(
            `openApp.jyingApp://virtual?params={"category":"jump","des":"${des}", "params": ${JSON.stringify(params)}}`
        )
    }
```
